## Installation

1. Open Audion.
2. Go to **Settings > Plugins**.
3. Click **Open Plugin Folder**.
4. Download or clone this plugin into the `plugins` directory.
   - Folder name should be `theme-customizer`.
5. Restart Audion or click **Reload Plugins**.
6. Enable the plugin in the settings menu.

## Usage

1. **Open Customizer**: Click the **Palette Icon** ðŸŽ¨ in the player bar or sidebar menu.
2. **Select a Theme**: Click on any of the cards in the **Presets** tab to apply a theme instantly.
3. **Customize**: Go to the **Custom** tab to tweak specific colors.
   - Use the color pickers to change Backgrounds, Accents, and Text colors.
4. **Save**: Click **Export / Save** to keep your creation.

## Advanced: CSS Injection

Some themes include custom CSS rules (like the animated gradients in *Aurora Borealis* or the scanlines in *Cyberpunk*). The plugin automatically injects these styles safely into the application.

## Permissions

This plugin requires the following permissions:
- `ui:inject`: To apply styles and show the customizer panel.
- `storage:local`: To save your active theme preference and custom presets.
